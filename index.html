GeoCat Bridge for QGIS — community docs

This is the index page of the community documentation for the GeoCat Bridge QGIS plugin. The public Git repository for the software and the documentation source is hosted on GitHub.

Documentation is available for the following GeoCat Bridge versions:

  • latest
  • v4.1
  • v4.0

If you are a using a development version of Bridge, you'll probably need the "latest" version (i.e. built from the HEAD Git commit).
